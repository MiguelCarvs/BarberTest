# CutsBarbier

Protótipo simples de um sistema de agendamento para barbearia, desenvolvido com HTML, CSS e JavaScript puro.

## Sobre o projeto

O projeto simula uma interface moderna de uma barbearia, contendo:

- Página inicial com tipos de corte
- Cards interativos
- Navegação entre páginas
- Página de agendamento
- Calendário visual
- Seleção de horários
- Modal de confirmação de agendamento
- Responsividade para dispositivos móveis

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

### Página inicial
- Exibição dos serviços
